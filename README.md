# Pinger

Простой сервис для поддержания активности приложений на Render через периодический пинг.

## Деплой на Render
   - **Environment**: `Python`
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `python app.py`

## Переменные окружения

| Переменная | Описание | Значение по умолчанию |
|------------|----------|----------------------|
| `URLS_TO_PING` | URL-адреса для пинга через запятую | (обязательная) |
| `PING_INTERVAL` | Интервал пинга в секундах | `30` |
| `PORT` | Порт сервера | `10000` |

## Эндпоинты

- `GET /` - статус сервиса и конфигурация
- `GET /ping` - ручной пинг всех URL

# Установить интервал 5 минут
